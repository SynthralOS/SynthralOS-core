## ⚠️ IMPORTANT: Values You MUST Update

Before deploying, you **MUST** update these values:

1. **NANGO_SECRET_KEY** - Get from [nango.dev](https://nango.dev) (REQUIRED for OAuth connectors)
2. **OPENAI_API_KEY** - Get from [platform.openai.com](https://platform.openai.com) (if using OpenAI)
3. **ANTHROPIC_API_KEY** - Get from [console.anthropic.com](https://console.anthropic.com) (if using Anthropic)
4. **RESEND_API_KEY** - Get from [resend.com](https://resend.com) (replace `re_xxxxx`)

## How to Add in Render

1. Go to your service (e.g., `sos-backend`)
2. Click **"Environment"** tab
3. For each variable above:
   - Click **"Add Environment Variable"**
   - **Key**: `DATABASE_URL`
   - **Value**: `postgresql://postgres:SynthralOS@db.qgfutvkhhsjbjthkammv.supabase.co:5432/postgres`
   - Click **"Save Changes"**
4. Repeat for each variable
